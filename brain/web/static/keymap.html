Abilities role → key the Defiler presses

Key format: a bare key (4, f2) or Mod+Key (Ctrl+1, Alt+=). Empty = unmapped. mode: auto = the loop may fire it; manual = dashboard button only.

Group & targeting per-slot character name + target key

To act on a member (heal / ward / cure / revive) the agent taps the target key, then the ability key. EQ2 default: F1 self, F2–F6 members. The character name is used for combat detection (the tank's name is the key one — change it when you swap who you're partnered with). Targeting is by SLOT, not name.
